# Vim function summary

## Multiple windows

* Ctrl-W, Ctrl-j,k,l,h to move the focus
* :sp, :vs create new windows
* Ctrl-w, Ctrl-w change to next window
* Ctrl-w, c close current windows

## Other settings

* :set sw=4 indent is 4 
* :set ts=4 tab=4
* :syntax on syntax is on
